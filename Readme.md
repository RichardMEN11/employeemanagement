## Quick Start

Prerequisites:

- Docker installed and running
- yarn installed

```bash
cd backend
docker-compose up
cd ../frontend
yarn run dev
```

After the start the backend is available under http://localhost:3000 and the frontend under http://localhost:8080.

## Technologies used

- Frontend - Nextjs
- Backend - Nestjs
- Database - Postgres
